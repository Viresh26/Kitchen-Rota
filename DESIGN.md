# Design System

## Theme Overview
This document outlines the core visual theme of our design system.

## Color Palette
The color palette is designed to be vibrant and accessible, ensuring a consistent visual experience across all products.

*   **Primary Color**: `#1275e2` - This is the brand's most distinctive chromatic color, used for key interactive elements, buttons, and calls to action.
*   **Secondary Color**: `#5f78a3` - A supporting color for less prominent UI elements, such as chips, secondary actions, and subtle accents.
*   **Tertiary Color**: `#c55b00` - An additional accent color for highlights, badges, or decorative elements, providing visual interest.
*   **Neutral Color**: `#74777f` - A foundational neutral color for backgrounds, surfaces, text, and non-chromatic elements, ensuring readability and visual balance.

The current color mode is set to **light**.

## Typography
Our typography system uses a consistent font family across various text roles to maintain harmony and legibility.

*   **Headline Font**: `Inter` - Used for all headings and prominent display text.
*   **Body Font**: `Inter` - Applied to all paragraph text and general content.
*   **Label Font**: `Inter` - Utilized for form labels, small text, and UI annotations.

## Shape and Spacing
The visual characteristics of UI elements are defined by their roundedness and the spacing between them.

*   **Roundedness**: `2` (Moderate) - UI elements feature a moderate level of rounded corners, providing a balanced and friendly aesthetic.
*   **Spacing**: `2` (Normal) - The spacing between and within UI elements is set to a normal level, ensuring a comfortable and clear layout without feeling too dense or too sparse.
